\# Car Price Prediction 🚗💰



This project uses Machine Learning to predict the selling price of cars based on features like year, fuel type, kilometers driven, and transmission.



\## 🎯 Objective

To build a regression model that estimates car prices.



\## ⚙️ Process

\- Data preprocessing using pandas  

\- Encoding categorical variables  

\- Train-test split  

\- Model training using Linear Regression  



\## 📈 Result

The model predicts car prices in lakhs with good accuracy.



\## 🛠️ Tools Used

\- Python  

\- Pandas  

\- Scikit-learn  

\- Jupyter Notebook  



\## ▶️ How to Run

1\. Open the notebook  

2\. Run all cells  

3\. View predicted price output
