# 🚗 Car Price Prediction 💰

✨ A Machine Learning project to predict car selling prices.

---

## 🎯 Objective
Build a regression model to estimate car prices.

---

## ⚙️ Process
- Data preprocessing using pandas  
- Encoding categorical variables  
- Train-test split  
- Model training using Linear Regression  

---

## 📈 Result
✔ Predicts car prices in **lakhs**  
✔ Provides accurate estimation  

💡 Example:
> Predicted Price ≈ **3.94 Lakhs**

---

## 🛠️ Tools Used
- Python 🐍  
- Pandas 📊  
- Scikit-learn 🤖
