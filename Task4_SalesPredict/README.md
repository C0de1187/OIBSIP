# 📊 Sales Prediction using Machine Learning 📈

This project predicts product sales based on advertising expenditure using Machine Learning.

---

## 🎯 Objective
To build a regression model that estimates sales using advertising data such as TV, Radio, and Newspaper.

---

## ⚙️ Process
- Loaded and explored the dataset  
- Cleaned data and removed unnecessary columns  
- Visualized relationships using Matplotlib  
- Split data into training and testing sets  
- Trained model using Linear Regression  
- Evaluated model using R² Score  

---

## 🧠 Concepts Used
- Regression  
- Linear Regression  
- Data Visualization  
- Model Evaluation (R² Score)  

---

## 📈 Result
The model predicts sales values based on advertising inputs with good accuracy.

💡 Example:
> Predicted Sales ≈ 22.5

---

## 🛠️ Tools Used
- Python 🐍  
- Pandas 📊  
- NumPy 🔢  
- Matplotlib 📈  
- Scikit-learn 🤖  

---

## ▶️ How to Run
1. Open the notebook (`sales_model.ipynb`)  
2. Run all cells  
3. View predictions
