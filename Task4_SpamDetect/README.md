# 📩 Spam Detection using Machine Learning 🚫

This project uses Machine Learning and Natural Language Processing (NLP) to classify messages as **Spam** or **Not Spam (Ham)**.

---

## 🎯 Objective
To build a model that can automatically detect whether a message is spam or not.

---

## ⚙️ Process
- Loaded and cleaned the dataset  
- Converted text data into numerical form using **CountVectorizer**  
- Split data into training and testing sets  
- Trained model using **Naive Bayes algorithm**  
- Evaluated model using accuracy  

---

## 🧠 Concepts Used
- Natural Language Processing (NLP)  
- CountVectorizer  
- Naive Bayes  
- Classification  

---

## 📈 Result
The model successfully classifies messages as spam or not spam with good accuracy.

💡 Example:
> “Congratulations! You won a prize” → **Spam 🚫**
