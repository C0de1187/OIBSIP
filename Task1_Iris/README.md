# Iris Flower Classification 🌸

This project uses Machine Learning to classify iris flowers into three species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

## 📊 Dataset
The dataset contains the following features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## ⚙️ Steps Performed
- Loaded dataset using pandas
- Cleaned data
- Split into training and testing sets
- Trained model using KNN algorithm
- Evaluated model using accuracy

## 🎯 Result
The model successfully predicts the species of iris flowers with high accuracy.

- Python
- Pandas
- Scikit-learn
- Jupyter Notebook
